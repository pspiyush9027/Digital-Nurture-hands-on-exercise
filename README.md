# Digital-Nurture-hands-on-exercise

Hands-on exercises for Java, design patterns, testing, logging, Spring, and PL/SQL.

## Modules

| Module | Folder | Description |
|--------|--------|-------------|
| Design Patterns | `Design-principal-and-pattern/` | Adapter, Builder, Factory Method, Singleton, MVC, Dependency Injection |
| Algorithms & Data Structures | `Algorithms-Data_Structures/` | E-Commerce product search, Financial forecasting |
| TDD (JUnit & Mockito) | `TDD_using_Junit_and_Mockito/` | JUnit basics, Mockito mocking |
| SLF4J Logging | `SL4J_Logging_Exercises/` | Logging levels and error messages |
| Spring Core | `Spring_Core_Maven/` | XML-based Spring application configuration |
| PL/SQL | `PLSQL_Exercises/` | Control structures, stored procedures |

## Project Types

- **IntelliJ modules** — open subfolder in IntelliJ and run `Main.java`
- **Maven projects** — run `mvn compile` then `mvn test` or `mvn exec:java`
