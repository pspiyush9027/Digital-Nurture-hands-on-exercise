# Blog App - React Lifecycle Hooks

This exercise uses a class component to demonstrate React lifecycle methods.

- `componentDidMount()` calls `loadPosts()` after the component is added to the page.
- `loadPosts()` fetches posts from `https://jsonplaceholder.typicode.com/posts`.
- `componentDidCatch()` displays an alert if React catches a rendering error.

Run the project:

```bash
npm install
npm run dev
```

Open the URL shown by Vite, normally `http://localhost:3000`.
